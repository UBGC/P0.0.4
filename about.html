PLATE-O-GUESSR is a computer game based on GE0-GUESSR, made on programming languages like, Javascript, HTML and more, by polish teenager.
That game is new, so expect bugs.
If you notice any bug, please contact with developer.

Tutorial

PLATE-O-GUESSR is a game in which you have to guess the place based on the car license plate.
You need to mark the closest place, from which comes that license plate
As close as you marked, you get more points.
After marking, click the check button.
Good luck!
